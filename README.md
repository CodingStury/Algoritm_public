# 스터디 진행 방식

1. 코테 문제를 하나 정한 후, 스터디 전까지 푼다.
2. 스터디가 시작되면 서로의 풀이 방법을 공유 및 리뷰한다.
3. 리뷰가 끝나면, 다음 스터디 전까지 풀어올 코테 문제를 정한다.
4. 스터디 끝.
